# XNWYMRM_self_define_control
此内容用于对RM机器人的控制方式经行自定义

## ReadMe
connect_direct.py 为开发者文档中直连的内容
connect_wifi.py 为开发者文档中wifi连接的内容
get_msg.py 为直连模式，用于测试赛事引擎数据
get_IP.py 为wifi模式获取机器IP地址
